# Documentation screenshots

These images are referenced by the guide pages under `src/content/docs/guides/`.
Capture them from the app (light mode) and save each with the exact filename
below into this folder. The pages will then render them automatically — no doc
edits needed.

Capture at a consistent window size (≈1440×900 works well) for a uniform look.
Local dev URLs use the mock workspace `nweb-corp:mock-ws-1`; on production use
the equivalent screens.
