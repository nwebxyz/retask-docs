A **task** is a unit of work inside a project. Tasks have a title, description, status, priority, assignees, and an optional due date.

Document in this page:
- Creating a task and the available fields
- Assigning tasks to workspace members
- Status, priority, and filtering
- Task keys (e.g. ENG-42) and attachments
